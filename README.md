## nix-shell-bit

Launch a nix-shell for the current project.
```sh
nix-shell-bit
```

## development

build
```sh
make
```

run tests
```sh
make test
```

integration tests
```sh
make integration-test
```

generate completion scripts
```sh
make completions
```

install
```sh
make install
```

uninstall
```sh
make uninstall
```
